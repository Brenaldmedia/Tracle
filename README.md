## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Brenaldmedia/Tracle.git
cd Tracle

# Install dependencies
npm install

# Start the bot
node index.js
First run will show a QR code or pairing code for login. Sessions are saved automatically.
🚀 Quick Commands
.menu — Show all commands
.tagall — Mention everyone
.imagine <prompt> — Generate AI image
.tiktok <url> — Download TikTok video
.status on — Enable auto view status
