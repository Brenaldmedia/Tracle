🛠️ Installation

Clone & Install

$ git clone https://github.com/Brenaldmedia/Tracle.git
$ cd Tracle
$ npm install

Run the Bot

$ node index.js

On first run, Tracle will ask for your WhatsApp number and send you a login code. Enter the code and you're connected.

🌍 Deployment Notes

Local / VPS

Works out of the box with interactive login (number + OTP).

Render / Railway / Heroku

Planned in v1.2.0 via persistent session storage (no terminal input needed).
